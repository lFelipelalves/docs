---
title: "Inboxes (canais)"
description: "Gestão dos canais de atendimento, horário comercial, agentes e bots."
---

Uma **inbox** representa um canal de comunicação conectado à plataforma. Cada inbox
tem agentes atribuídos, configurações próprias (horário, mensagem fora de expediente,
auto-resolve) e regras de roteamento.

**Base:** `https://chat.lfautomatiza.com/api/v1/accounts/{account_id}`

---

## Criar inbox

```http
POST /inboxes
```

<ParamField body="name" type="string" required>Nome da inbox</ParamField>
<ParamField body="channel" type="object" required>Configuração do canal (campos variam por `type`)</ParamField>

#### Exemplo `channel` para WhatsApp via API channel

```json
{
  "type": "api",
  "webhook_url": "https://seu-webhook.com/inbox-events"
}
```

#### Exemplo `channel` para Webchat

```json
{
  "type": "web_widget",
  "website_url": "https://lfautomatiza.com",
  "welcome_title": "Como podemos ajudar?",
  "welcome_tagline": "Resposta em até 5 minutos",
  "widget_color": "#00B86B"
}
```

### Requisição

```bash cURL
curl -X POST https://chat.lfautomatiza.com/api/v1/accounts/1/inboxes \
  -H "api_access_token: $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Site - Webchat",
    "channel": {
      "type": "web_widget",
      "website_url": "https://lfautomatiza.com",
      "welcome_title": "Como podemos ajudar?",
      "widget_color": "#00B86B"
    }
  }'
```

### Resposta `200`

```json
{
  "id": 4,
  "name": "Site - Webchat",
  "channel_type": "Channel::WebWidget",
  "website_token": "xpto1234abcd",
  "widget_color": "#00B86B"
}
```

## Atualizar inbox

```http
PATCH /inboxes/{id}
```

<ParamField path="id" type="integer" required>ID da inbox</ParamField>
<ParamField body="name" type="string">Nome da inbox</ParamField>
<ParamField body="greeting_enabled" type="boolean">Liga mensagem de saudação inicial</ParamField>
<ParamField body="greeting_message" type="string">Texto da saudação</ParamField>
<ParamField body="working_hours_enabled" type="boolean">Liga horário comercial</ParamField>
<ParamField body="out_of_office_message" type="string">Mensagem fora do expediente</ParamField>
<ParamField body="csat_survey_enabled" type="boolean">Dispara CSAT ao resolver</ParamField>
<ParamField body="enable_auto_assignment" type="boolean">Distribuição automática round-robin</ParamField>
<ParamField body="auto_assignment_config" type="object">`{ max_assignment_limit }` — máximo de conversas por agente</ParamField>
<ParamField body="allow_messages_after_resolved" type="boolean">Cliente pode mandar msg após resolvida (reabre)</ParamField>
<ParamField body="working_hours" type="array">Configuração de horário (ver seção abaixo)</ParamField>
<ParamField body="timezone" type="string">Fuso (`America/Sao_Paulo`)</ParamField>

### Requisição

```bash cURL
curl -X PATCH https://chat.lfautomatiza.com/api/v1/accounts/1/inboxes/1 \
  -H "api_access_token: $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "greeting_enabled": true,
    "greeting_message": "Olá! Como podemos ajudar?",
    "enable_auto_assignment": true,
    "auto_assignment_config": { "max_assignment_limit": 5 }
  }'
```

### Resposta `200`

```json
{
  "id": 1,
  "greeting_enabled": true,
  "greeting_message": "Olá! Como podemos ajudar?",
  "enable_auto_assignment": true,
  "auto_assignment_config": { "max_assignment_limit": 5 },
  "updated_at": "2026-05-18T20:00:00Z"
}
```

## Configurar horário comercial

```http
PATCH /inboxes/{id}
```

Envie o payload com `working_hours` no formato abaixo. `day_of_week`: 0=Domingo, 1=Segunda... 6=Sábado.

### Requisição

```bash cURL
curl -X PATCH https://chat.lfautomatiza.com/api/v1/accounts/1/inboxes/1 \
  -H "api_access_token: $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "timezone": "America/Sao_Paulo",
    "working_hours_enabled": true,
    "out_of_office_message": "Nosso horário é seg-sex, 9h-18h.",
    "working_hours": [
      { "day_of_week": 0, "closed_all_day": true },
      { "day_of_week": 1, "open_hour": 9, "open_minutes": 0, "close_hour": 18, "close_minutes": 0 },
      { "day_of_week": 2, "open_hour": 9, "open_minutes": 0, "close_hour": 18, "close_minutes": 0 },
      { "day_of_week": 3, "open_hour": 9, "open_minutes": 0, "close_hour": 18, "close_minutes": 0 },
      { "day_of_week": 4, "open_hour": 9, "open_minutes": 0, "close_hour": 18, "close_minutes": 0 },
      { "day_of_week": 5, "open_hour": 9, "open_minutes": 0, "close_hour": 18, "close_minutes": 0 },
      { "day_of_week": 6, "closed_all_day": true }
    ]
  }'
```

### Resposta `200`

```json
{
  "id": 1,
  "working_hours_enabled": true,
  "timezone": "America/Sao_Paulo",
  "working_hours": [
    { "day_of_week": 0, "closed_all_day": true },
    { "day_of_week": 1, "open_hour": 9, "close_hour": 18 }
  ]
}
```

## Excluir inbox

```http
DELETE /inboxes/{id}
```

<ParamField path="id" type="integer" required>ID da inbox</ParamField>

<Warning>
  Excluir a inbox **apaga todas as conversas vinculadas**. Considere desconectar
  o canal em vez de excluir se quiser preservar histórico.
</Warning>

### Requisição

```bash cURL
curl -X DELETE https://chat.lfautomatiza.com/api/v1/accounts/1/inboxes/4 \
  -H "api_access_token: $TOKEN"
```

### Resposta `204`

Sem corpo.

## Adicionar / substituir agentes da inbox

```http
POST /inboxes/{id}/inbox_members
```

O array enviado **substitui** completamente a lista atual.

<ParamField path="id" type="integer" required>ID da inbox</ParamField>
<ParamField body="user_ids" type="array" required>IDs dos agentes que devem ter acesso</ParamField>

### Requisição

```bash cURL
curl -X POST https://chat.lfautomatiza.com/api/v1/accounts/1/inboxes/1/inbox_members \
  -H "api_access_token: $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{ "user_ids": [7, 11, 15] }'
```

### Resposta `200`

```json
{
  "inbox_id": 1,
  "members_count": 3,
  "user_ids": [7, 11, 15]
}
```

## Vincular / desvincular bot

```http
POST /inboxes/{id}/set_agent_bot
```

<ParamField path="id" type="integer" required>ID da inbox</ParamField>
<ParamField body="agent_bot" type="integer">ID do bot (passe `null` pra desvincular)</ParamField>

### Requisição — vincular

```bash cURL
curl -X POST https://chat.lfautomatiza.com/api/v1/accounts/1/inboxes/1/set_agent_bot \
  -H "api_access_token: $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{ "agent_bot": 4 }'
```

### Requisição — desvincular

```bash cURL
curl -X POST https://chat.lfautomatiza.com/api/v1/accounts/1/inboxes/1/set_agent_bot \
  -H "api_access_token: $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{ "agent_bot": null }'
```

### Resposta `200`

```json
{ "inbox_id": 1, "agent_bot": 4 }
```

## Tipos de canal (`channel_type`)

| `channel_type` | Canal |
| --- | --- |
| `Channel::Whatsapp` | WhatsApp |
| `Channel::FacebookPage` | Facebook Messenger |
| `Channel::Instagram` | Instagram DM |
| `Channel::WebWidget` | Webchat (widget no site) |
| `Channel::Email` | E-mail |
| `Channel::Telegram` | Telegram |
| `Channel::Sms` | SMS |
| `Channel::Api` | Canal API customizado |

---

## Códigos de resposta

| Código | Quando acontece |
| --- | --- |
| `200 OK` | Sucesso em GET / POST / PATCH |
| `201 Created` | Inbox criada |
| `204 No Content` | Inbox excluída ou agentes removidos |
| `400 Bad Request` | Payload inválido |
| `403 Forbidden` | Apenas administrador pode gerenciar inboxes |
| `404 Not Found` | Inbox não existe |
| `422 Unprocessable Entity` | Configuração de canal inválida ou agentes não pertencem à conta |
