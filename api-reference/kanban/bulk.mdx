Para agir sobre dezenas ou centenas de cards de uma vez, use os endpoints de
**ações em massa** em vez de fazer N chamadas individuais — é mais rápido e
respeita o rate limit.

**Base:** `https://chat.lfautomatiza.com/api/v1/accounts/{account_id}`

---

## Mover cards em massa

```http
POST /kanban_items/bulk_move_items
```

<ParamField body="item_ids" type="array" required>IDs dos cards a mover</ParamField>
<ParamField body="new_stage" type="string" required>Etapa destino</ParamField>
<ParamField body="funnel_id" type="integer">Funil destino (se diferente do atual)</ParamField>

### Requisição

```bash cURL
curl -X POST https://chat.lfautomatiza.com/api/v1/accounts/1/kanban_items/bulk_move_items \
  -H "api_access_token: $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "item_ids": [101, 102, 103, 104],
    "new_stage": "proposal",
    "funnel_id": 10
  }'
```

### Resposta `200`

```json
{
  "moved_count": 4,
  "items": [
    { "id": 101, "funnel_stage": "proposal" },
    { "id": 102, "funnel_stage": "proposal" },
    { "id": 103, "funnel_stage": "proposal" },
    { "id": 104, "funnel_stage": "proposal" }
  ],
  "automations_triggered": 12
}
```

## Busca textual

```http
GET /kanban_items/search
```

Busca em título, descrição e conteúdo de notas.

<ParamField query="query" type="string" required>Texto a buscar</ParamField>
<ParamField query="funnel_id" type="integer">Limita a um funil específico</ParamField>
<ParamField query="agent_id" type="integer">Limita a cards atribuídos a um agente</ParamField>

### Requisição

```bash cURL
curl "https://chat.lfautomatiza.com/api/v1/accounts/1/kanban_items/search?query=ACME&funnel_id=10" \
  -H "api_access_token: $TOKEN"
```

### Resposta `200`

```json
{
  "query": "ACME",
  "matches": [
    {
      "id": 101,
      "item_details": { "title": "Oportunidade ACME", "value": 5000 },
      "funnel_stage": "qualified",
      "highlight": ["title"]
    }
  ],
  "count": 1
}
```

## Filtro avançado

```http
GET /kanban_items/filter
```

<ParamField query="funnel_id" type="integer">Limita a um funil</ParamField>
<ParamField query="priorities[]" type="array">Uma ou mais prioridades (`low`, `medium`, `high`, `urgent`)</ParamField>
<ParamField query="value_min" type="number">Valor mínimo</ParamField>
<ParamField query="value_max" type="number">Valor máximo</ParamField>
<ParamField query="agent_id" type="integer">Agente atribuído</ParamField>
<ParamField query="from" type="date">Início do intervalo de criação</ParamField>
<ParamField query="to" type="date">Fim do intervalo</ParamField>

### Requisição

```bash cURL
curl "https://chat.lfautomatiza.com/api/v1/accounts/1/kanban_items/filter?funnel_id=10&priorities[]=high&priorities[]=urgent&value_min=1000&value_max=50000&agent_id=7&from=2026-05-01&to=2026-05-31" \
  -H "api_access_token: $TOKEN"
```

### Resposta `200`

```json
{
  "filters_applied": {
    "funnel_id": 10,
    "priorities": ["high", "urgent"],
    "value_range": [1000, 50000],
    "agent_id": 7,
    "date_range": ["2026-05-01", "2026-05-31"]
  },
  "items": [/* cards */],
  "count": 18,
  "total_value": 87500
}
```

## Exportar CSV

```http
GET /kanban_items/export
```

<ParamField query="funnel_id" type="integer" required>Funil a exportar</ParamField>
<ParamField query="stage_id" type="string">Filtra por etapa (opcional)</ParamField>
<ParamField query="from" type="date">Início (opcional)</ParamField>
<ParamField query="to" type="date">Fim (opcional)</ParamField>

CSV vem em **UTF-8 com BOM** e separador `;` — abre direto no Excel BR.

### Requisição

```bash cURL
curl "https://chat.lfautomatiza.com/api/v1/accounts/1/kanban_items/export?funnel_id=10" \
  -H "api_access_token: $TOKEN" \
  -o cards.csv
```

### Resposta `200`

```csv
id;titulo;valor;moeda;etapa;agente;criado_em;atualizado_em
101;Oportunidade ACME;5000;BRL;qualified;Sabrina;2026-05-10T10:00:00Z;2026-05-18T14:00:00Z
102;Beta Corp;7500;BRL;proposal;Marcos;2026-05-12T08:00:00Z;2026-05-18T12:00:00Z
```

## Preview de importação

```http
POST /kanban_items/import_preview
```

Multipart. Retorna as primeiras 10 linhas detectadas + colunas identificadas, pra
você mapear no próximo passo.

<ParamField body="file" type="file" required>Arquivo CSV (multipart)</ParamField>

### Requisição

```bash cURL
curl -X POST https://chat.lfautomatiza.com/api/v1/accounts/1/kanban_items/import_preview \
  -H "api_access_token: $TOKEN" \
  -F "file=@leads.csv"
```

### Resposta `200`

```json
{
  "detected_columns": ["Nome", "Empresa", "Valor", "Telefone"],
  "preview_rows": [
    { "Nome": "Carlos", "Empresa": "ACME", "Valor": "5000", "Telefone": "+55..." },

  "total_rows_detected": 1247,
  "suggested_mappings": {
    "Nome": "title",
    "Empresa": "description",
    "Valor": "value"
  }
}
```

## Importar definitivamente

```http
POST /kanban_items/import
```

<ParamField body="file" type="file" required>Arquivo CSV</ParamField>
<ParamField body="funnel_id" type="integer" required>Funil destino</ParamField>
<ParamField body="default_stage_id" type="string">Etapa onde criar os cards (padrão: primeira do funil)</ParamField>
<ParamField body="mappings" type="object" required>Mapa `coluna_csv → campo_da_api` em JSON</ParamField>

### Requisição

```bash cURL
curl -X POST https://chat.lfautomatiza.com/api/v1/accounts/1/kanban_items/import \
  -H "api_access_token: $TOKEN" \
  -F "file=@leads.csv" \
  -F "funnel_id=10" \
  -F "default_stage_id=lead" \
  -F 'mappings={"Nome":"title","Empresa":"description","Valor":"value"}'
```

### Resposta `200`

```json
{
  "import_id": "imp_abc123",
  "status": "processing",
  "total_rows": 1247,
  "estimated_seconds": 90
}
```

<Tip>
  Pra importação em produção, sempre rode `import_preview` antes pra confirmar
  o mapeamento de colunas. Importação em massa não tem desfazer fácil.
</Tip>

## Códigos de resposta

| Código | Quando acontece |
| --- | --- |
| `200 OK` | Operação concluída |
| `204 No Content` | Importação enfileirada (processamento assíncrono) |
| `400 Bad Request` | `item_ids` vazio, `funnel_id` ausente, CSV malformado |
| `403 Forbidden` | Sem permissão no funil |
| `413 Payload Too Large` | Arquivo CSV maior que limite |
| `422 Unprocessable Entity` | Mapping inválido, etapa inexistente, prioridade desconhecida |
