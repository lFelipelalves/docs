Pra agir sobre dezenas ou centenas de cards de uma vez, use os endpoints de
**ações em massa** em vez de fazer N chamadas individuais — é mais rápido e
respeita o rate limit.

## Ações em massa

| Método | Endpoint | Descrição |
| --- | --- | --- |
| `POST` | `/kanban_items/bulk_move_items` | Move múltiplos cards de etapa |
| `POST` | `/kanban_items/bulk_assign_agent` | Atribui agente em lote |
| `POST` | `/kanban_items/bulk_set_priority` | Atualiza prioridade em lote |

### Mover em massa

```bash
curl -X POST https://chat.lfautomatiza.com/api/v1/accounts/1/kanban_items/bulk_move_items \
  -H "api_access_token: $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "item_ids": [101, 102, 103, 104],
    "new_stage": "proposal",
    "funnel_id": 10
  }'
```

### Atribuir agente em lote

```bash
curl -X POST https://chat.lfautomatiza.com/api/v1/accounts/1/kanban_items/bulk_assign_agent \
  -H "api_access_token: $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "item_ids": [101, 102, 103],
    "agent_id": 7,
    "mode": "replace"
  }'
```

| `mode` | Comportamento |
| --- | --- |
| `replace` | Substitui agentes atuais pelo `agent_id` |
| `add` | Adiciona o `agent_id` aos agentes já atribuídos |

### Mudar prioridade em lote

```bash
curl -X POST https://chat.lfautomatiza.com/api/v1/accounts/1/kanban_items/bulk_set_priority \
  -H "api_access_token: $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{ "item_ids": [101, 102, 103], "priority": "high" }'
```

## Busca e filtros avançados

| Método | Endpoint | Descrição |
| --- | --- | --- |
| `GET` | `/kanban_items/search` | Busca textual em título/descrição/notas |
| `GET` | `/kanban_items/filter` | Filtros avançados (valor, datas, prioridade) |
| `GET` | `/kanban_items/reports` | Métricas agregadas |
| `GET` | `/kanban_items/debug` | Dados técnicos (debug, suporte) |

### Filtro avançado

```bash
curl "https://chat.lfautomatiza.com/api/v1/accounts/1/kanban_items/filter?funnel_id=10&priorities[]=high&priorities[]=urgent&value_min=1000&value_max=50000&agent_id=7&from=2026-05-01&to=2026-05-31" \
  -H "api_access_token: $TOKEN"
```

Parâmetros suportados:

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| `funnel_id` | integer | Filtra por funil |
| `priorities[]` | array | Múltiplas prioridades simultâneas |
| `value_min` | number | Valor mínimo do card |
| `value_max` | number | Valor máximo |
| `agent_id` | integer | Atribuído a este agente |
| `from` / `to` | date | Intervalo de criação |

### Relatórios

```bash
curl "https://chat.lfautomatiza.com/api/v1/accounts/1/kanban_items/reports?funnel_id=10&from=2026-05-01&to=2026-05-31&user_ids[]=7&user_ids[]=11&inbox_id=1" \
  -H "api_access_token: $TOKEN"
```

Retorna: total por etapa, conversion rate entre etapas, tempo médio por etapa,
valor total ganho/perdido no período.

## Importação e exportação CSV

| Método | Endpoint | Descrição |
| --- | --- | --- |
| `GET` | `/kanban_items/export` | Exporta CSV |
| `POST` | `/kanban_items/import_preview` | Pré-visualiza CSV antes de importar |
| `POST` | `/kanban_items/import` | Importa CSV definitivamente |

### Exportar

```bash
curl "https://chat.lfautomatiza.com/api/v1/accounts/1/kanban_items/export?funnel_id=10" \
  -H "api_access_token: $TOKEN" \
  -o cards.csv
```

CSV vem em **UTF-8 com BOM** e separador `;` — abre direto no Excel BR sem
quebrar acentuação.

### Preview de importação

```bash
curl -X POST https://chat.lfautomatiza.com/api/v1/accounts/1/kanban_items/import_preview \
  -H "api_access_token: $TOKEN" \
  -F "file=@leads.csv"
```

Retorna as primeiras 10 linhas detectadas + colunas identificadas, pra você
mapear no próximo passo.

### Importar definitivamente

```bash
curl -X POST https://chat.lfautomatiza.com/api/v1/accounts/1/kanban_items/import \
  -H "api_access_token: $TOKEN" \
  -F "file=@leads.csv" \
  -F "funnel_id=10" \
  -F "default_stage_id=lead" \
  -F 'mappings={"Nome":"title","Empresa":"description","Valor":"value"}'
```

<Tip>
  Pra importação em produção, sempre rode `import_preview` antes pra confirmar
  o mapeamento de colunas. Importação em massa não tem desfazer fácil.
</Tip>
