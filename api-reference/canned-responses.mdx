---
title: "Respostas rápidas"
description: "Textos pré-definidos que o agente insere com atalho durante o atendimento."
---

Respostas rápidas (canned responses) são templates de texto que o agente acessa
digitando `/` na caixa de mensagem. Diferente de macros, **não executam ações** —
apenas inserem texto.

**Base:** `https://chat.lfautomatiza.com/api/v1/accounts/{account_id}`

---

## Listar respostas

```http
GET /canned_responses
```

### Requisição

```bash cURL
curl https://chat.lfautomatiza.com/api/v1/accounts/1/canned_responses \
  -H "api_access_token: $TOKEN"
```

### Resposta `200`

```json
[
  {
    "id": 41,
    "short_code": "saudacao",
    "content": "Olá! Tudo bem? Como posso te ajudar hoje?",
    "created_at": "2026-04-12T10:00:00Z"
  },
  {
    "id": 42,
    "short_code": "horario",
    "content": "Nosso horário de atendimento é seg-sex, 9h-18h.",
    "created_at": "2026-04-12T10:05:00Z"
  }
]
```

## Criar resposta

```http
POST /canned_responses
```

<ParamField body="short_code" type="string" required>Atalho (`/saudacao`, `/boleto`). Sem espaço.</ParamField>
<ParamField body="content" type="string" required>Texto inserido quando o atalho é usado</ParamField>

### Variáveis dinâmicas no `content`

O `content` aceita placeholders substituídos na inserção:

| Variável | Valor |
| --- | --- |
| `{{contact.name}}` | Nome do contato |
| `{{contact.email}}` | E-mail |
| `{{contact.phone_number}}` | Telefone |
| `{{conversation.id}}` | ID da conversa |
| `{{conversation.display_id}}` | Display ID (ticket #) |
| `{{agent.name}}` | Nome do agente que está enviando |

### Requisição

```bash cURL
curl -X POST https://chat.lfautomatiza.com/api/v1/accounts/1/canned_responses \
  -H "api_access_token: $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "short_code": "encerramento",
    "content": "Foi um prazer te atender, {{contact.name}}! Se precisar de algo mais, é só chamar. — {{agent.name}}"
  }'
```

### Resposta `201`

```json
{
  "id": 43,
  "short_code": "encerramento",
  "content": "Foi um prazer te atender, {{contact.name}}! Se precisar de algo mais, é só chamar. — {{agent.name}}",
  "created_at": "2026-05-19T00:15:00Z"
}
```

## Atualizar resposta

```http
PATCH /canned_responses/{id}
PUT /canned_responses/{id}
```

<ParamField path="id" type="integer" required>ID da resposta</ParamField>

### Requisição

```bash cURL
curl -X PATCH https://chat.lfautomatiza.com/api/v1/accounts/1/canned_responses/41 \
  -H "api_access_token: $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "content": "Olá {{contact.name}}, tudo bem? Como posso te ajudar?"
  }'
```

### Resposta `200`

```json
{
  "id": 41,
  "short_code": "saudacao",
  "content": "Olá {{contact.name}}, tudo bem? Como posso te ajudar?",
  "updated_at": "2026-05-19T00:30:00Z"
}
```

## Excluir resposta

```http
DELETE /canned_responses/{id}
```

<ParamField path="id" type="integer" required>ID da resposta</ParamField>

### Requisição

```bash cURL
curl -X DELETE https://chat.lfautomatiza.com/api/v1/accounts/1/canned_responses/43 \
  -H "api_access_token: $TOKEN"
```

### Resposta `204`

Sem corpo.

---

## Códigos de resposta

| Código | Quando acontece |
| --- | --- |
| `200 OK` | Listagem / leitura / atualização |
| `201 Created` | Resposta criada |
| `204 No Content` | Excluída |
| `422 Unprocessable Entity` | `short_code` duplicado na conta |
